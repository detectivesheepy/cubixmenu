# CUBIX
A Windows batch menu made for running commands, with admin bypass

## Run CUBIX
### CMD unblocked
To run CUBIX, run this command in the CMD: <br>
``set __COMPAT_LAYER=RUNASINVOKER
start "" "location"`` <br>
Replace `location` with the path to the CUBIX menu.
### CMD blocked
Simply double-click the file in the explorer. This may have the effect of not being able to use all commands though, as CUBIX relies on admin commands regularly.
